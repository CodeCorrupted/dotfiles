# Config of Rofi

[Rofi][1] is an application launcher. This config file only
 tells to Rofi the font and icons to use and some things
 about the display of the window. The theme used is the one
 provided by [Pywal][2].

## Installation

You just need to copy the file `config.rasi` to the `$HOME/.config/rofi/`
 directory. Also, be sure to have the following dependencies:

- [Hack Nerd Font][3].
- [Candy-icons][4].
- Pywal.

Now you can enjoy of Rofi running the following command:

```bash
rofi -show drun
```
