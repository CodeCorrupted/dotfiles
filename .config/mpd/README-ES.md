# Archivo de configuración MPD (Music Player Deamon)

Este archivo de configuración es para MPD y está hecho para funcionar con PulseAudio. Hecho y probado en Artix Linux, puede que no funcione correctamente en otras distros Linux.

Seguí las instrucciones de la [wiki de Arch](https://wiki.archlinux.org/title/Music_Player_Daemon) para este archivo y apliqué la configuración por usuario.

El cliente que uso para MPD es [ncmcpp](https://rybczak.net/ncmpcpp/).
